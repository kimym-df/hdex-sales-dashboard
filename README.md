# HDEX 매출 분석 대시보드

## 개요

HDEX 오프라인 매장 매출 분석을 위한 대시보드입니다.

- **로컬 경로:** `/Users/df_ym1/hdex-sales-dashboard/index.html`
- **데이터:** 2025년 엑셀 확정 / 2026년 Supabase

---

## 탭 구성

### 1. 매출 분석
- KPI: 총 매출, 연간 목표, 목표 달성률, 월평균 매출, 매장 수 등
- 월별 매출 추이 차트
- 매장별 매출 비교 / 비중 (막대, 도넛)
- 매장별 목표 달성률 (2026년)
- **매장별 월별 매출 상세** 테이블
- 매장별 월별 판매수량 상세
- 월별 매장 매출 순위

### 2. 일별 매출
- 2026년 일별 매출 테이블
- 가로: 매장명 / 세로: 일자 (2026-01-01 ~ Supabase 업데이트일)
- 2026년 데이터만 로딩 (속도 최적화)
